AIを神にしないためのメタ認知実践プロトコル公開サイトです。

このサイトは、魅力的な思想の宣伝よりも先に、以下の安全設計を公開するためのものです。

- AI出力の神託化を防ぐ

## GitHub Pages での使い方

1. ZIPを展開する
2. 中身のファイルをリポジトリ直下に置く
3. GitHub Pages を有効化する
4. トップページには `index.html` が表示される
