# 90分ワークショップ草案

## 目的

団体化、参加募集、有料化、ログ収集の前に、知性の鏡会プロトコルが安全に機能するかを確認するための最小実験です。

## 条件

- 成人のみ
- 少人数
- 無料
- 録画なし
- 録音なし
- AI対話ログ収集なし
- 個人相談なし
- 寄附なし
- 脱退自由
- 緊急相談不可
- 終了後アンケートのみ

## 当日の流れ

| 時間 | 内容 |
|---|---|
| 00:00-00:10 | 趣旨説明 |
| 00:10-00:20 | AI非絶対化説明 |
| 00:20-00:45 | 三AI問答 |
| 00:45-01:10 | プロンプト・レビュー |
| 01:10-01:20 | 沈黙の間 |
| 01:20-01:30 | 振り返り |

## 禁止事項

- AI出力を神託として扱うこと
- 創始者・運営を最終権威として扱うこと
- 人格攻撃、差別、屈辱の強制
- 自傷他害、違法行為、犯罪の助長
- 医療、法律、投資、精神危機の判断を共同体内で完結させること
- 未成年への個別接触
- 寄附や会費の勧誘

## 参加を見送るべき状態

- 睡眠不足や強い疲労がある
- 精神的に不安定である
- 自傷他害の危険がある
- AIや共同体に救済を求めている
- 医療、法律、投資などの専門判断を求めている
- 現実の人間関係、学業、仕事を避ける目的で参加しようとしている

## 記録方針

外部レビューに必要な運営記録のみ残します。個人のAI対話ログ、相談内容、録画、録音は収集しません。
