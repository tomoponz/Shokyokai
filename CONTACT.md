# 連絡先・責任主体の設定方針

## 現在の状態

正式な連絡窓口と責任主体は未設定です。

公開前に、以下を設定してください。

- 管理責任者または責任主体
- 問い合わせ用メールアドレス
- 批判・提案の受付方法
- 削除依頼・個人情報問い合わせの窓口
- 外部レビュー提案の受付方法

## 推奨される窓口

- 専用メールアドレス
- GitHub Issues
- 外部レビュー用の公開フォーム

## 注意

個人用メールアドレスを直接公開する場合は、迷惑メール、個人情報対応、責任範囲を考慮してください。

このプロジェクトは個人相談窓口ではありません。医療、法律、投資、精神危機、自傷他害、家庭内暴力、犯罪、依存症、未成年保護問題は専門機関を優先してください。
